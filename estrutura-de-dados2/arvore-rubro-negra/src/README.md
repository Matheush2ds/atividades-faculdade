# Comparativo de Desempenho entre Árvores AVL e Rubro-Negra

Este é um projeto de simulação em Java que compara o desempenho de duas árvores mágicas: a Árvore AVL e a Árvore Rubro-Negra.
## Resultados Simulados

### Preenchimento Inicial

1. **Árvore AVL:**
   - Tempo de Preenchimento: 54.72 segundos

2. **Árvore Rubro-Negra:**
   - Tempo de Preenchimento: 76.89 segundos
  
### Operações Adicionais

1. **Adaptação a Mudanças:**
   - **Árvore AVL:**
     - Tempo de Inserção Adicional: 82,78 segundos
   - **Árvore Rubro-Negra:**
     - Tempo de Inserção Adicional: 63.19 segundos

## Conclusões

- **Tempo de Execução:**
  - A Árvore AVL mostrou-se eficiente durante o preenchimento inicial.
  - A Árvore Rubro-Negra destacou-se em operações adicionais.

- **Adaptação a Mudanças:**
  - Ambas as árvores apresentaram características distintas na adaptação a mudanças.

- **Escolha da Estrutura:**
  - A escolha entre Árvore AVL e Árvore Rubro-Negra vai depender muito dos requisitos específicos do projeto.
